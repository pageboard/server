Pageboard Server
================

Install
-------

On linux, these packages are needed:

- postgresql database with unaccent extension
- libexiftool
- chrome or chromium
- libvips-dev
- libcurl-dev

See also `@pageboard/proxy`.

API
---

`npx pageboard --help`

To list all API modules.

`npx pageboard --help=<module.method>`

To list a specific API module.
